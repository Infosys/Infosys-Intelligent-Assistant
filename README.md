# Infosys-Intelligent-Assistant
Infosys Intelligent Assistant (IIA) is developed using advanced AI/ML capabilities to significantly increase the productivity of ticket management and resolution process.IIA is trained using historical ticket dump on machine learning algorithms to predict the various fields in the ticket.

In application maintenance services, teams perform several non-value-adding tasks manually which can lower the productivity of the entire process like reworking tickets after incorrect mapping, validating the information manually to enrich the ticket, expending efforts on known issues due to lack of visibility of related tickets and manual intervention for routine requests like account creation. IIA addresses all these challenges by automating the following:

•	Ticket classification in the right category

•	Assignment to the right group and SME

•	Validation and enrichment of details

•	Automated resolution using bots and scripts
